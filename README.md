# Priyanka Bhargav — Portfolio

Personal portfolio website built with Astro and Tailwind CSS.

## Tech Stack
- Astro
- Tailwind CSS
- JavaScript

## Live Site
[bhargavpriyanka.github.io/priyanka-bhargav-portfolio](https://bhargavpriyanka.github.io/priyanka-bhargav-portfolio)

## Run Locally
```bash
npm install
npm run dev
```

Open [http://localhost:4321/priyanka-bhargav-portfolio](http://localhost:4321/priyanka-bhargav-portfolio)
